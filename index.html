🗺️ Nouveau lieu
💡 Aide : comment analyser un lieu ?
Espace réel ou symbolique ?
La maison peut représenter la famille, la prison le déterminisme social, la nature la liberté.
Espace ouvert ou fermé ?
Un espace fermé crée un sentiment d'oppression ou de protection. Un espace ouvert suggère la liberté ou le danger.
Mouvements des personnages
Les déplacements révèlent une quête, une fuite, une progression intérieure.
Géographie symbolique
Ville / campagne · Haut / bas · Intérieur / extérieur — ces oppositions ont souvent un sens idéologique.
Questions à poser
Qui va dans ce lieu ? Pourquoi ? Qu'est-ce qui s'y passe d'important ? Qu'est-ce que ce lieu représente pour le personnage ?
💡 Aide : comment analyser le temps dans un récit ?
📐 L'Ordre — Chronologie vs anachronies
L'histoire est-elle racontée dans l'ordre ? Y a-t-il des retours en arrière (flashback / analepse) ou des anticipations (prolepse) ? → Effets : suspense, révélation progressive, surprise.
⏱️ La Durée — Rythme narratif
Quels moments sont dilatés (racontés lentement) ? Quels moments sont accélérés ou résumés en une phrase (ellipse) ? Certaines scènes s'arrêtent sur un détail (pause descriptive). → Effets : mise en valeur des moments importants.
🔁 La Fréquence — Répétitions
Certains événements sont-ils racontés une seule fois (singulatif), résumés en « il faisait souvent » (itératif), ou au contraire racontés plusieurs fois (répétitif) ? → Effets : insistance, habitude, obsession.
Questions à poser
Quelle est l'époque du récit ? La narration suit-elle l'ordre des événements ? Quels moments sont les plus longs à lire, et pourquoi ?

✍️ Ma Dissertation — Méthode 2nde
5 étapes : Analyse du sujet → Problématique → Plan → Rédaction → Relecture
Chaque paragraphe : Argument · Exemple · Analyse (méthode A.E.A.)
① Analyse du sujet
📌 Sujet
🔑 Mots-clés & définitions
Identifie les mots porteurs de sens. Donne plusieurs sens possibles à chaque terme.
🔄 Reformulation & tension
Reformule le sujet avec tes mots. Cherche l'opposition ou la nuance cachée.
② Problématique & Plan
❓ Problématique
Question centrale — ouverte, problématique, pas de réponse par oui/non. Ex. : « Dans quelle mesure… tout en… ? » / « En quoi… bien que… ? »
📋 Type de plan
Dialectique (thèse/antithèse/synthèse) · Analytique/thématique · Progressif
③ Introduction
🎯 Accroche
Citation, fait littéraire ou mise en contexte (2-3 lignes).
📖 Présentation de l'œuvre + définition + problématique + annonce
④ Développement — méthode A.E.A.
Chaque paragraphe = Argument (idée) → Exemple (citation + page) → Analyse (lien argument/exemple). C'est l'analyse qui compte le plus !
Partie I — Paragraphe 1
💡 Argument
📝 Exemple (citation + page ou résumé d'épisode)
🔍 Analyse (lien entre exemple et argument)
Partie II — Paragraphe 2
💡 Argument
📝 Exemple
🔍 Analyse
Partie III — Paragraphe 3 (facultatif)
💡 Argument
📝 Exemple
🔍 Analyse
⑤ Conclusion
📊 Bilan
Résume les grandes étapes du raisonnement en 2-3 lignes. Ne pas répéter — synthétiser.
✅ Réponse à la problématique
🔭 Ouverture
Perspective nouvelle : autre œuvre, enjeu actuel, question philosophique. Ne pas être artificiel.
💡 Connecteurs logiques
Introduire : D'abord • En premier lieu • Tout d'abord

🎭 Structure de l'intrigue

Choisis un modèle narratif, puis remplis chaque phase de l'histoire.

💡 À quoi servent ces modèles ?

Aristote (3 actes) : le plus simple — début, milieu, fin. Parfait pour un premier résumé.
Freytag (5 actes) : montre comment la tension monte puis redescend (pyramide).
Arc moderne (8 étapes) : le plus précis — donne un mot sur chaque moment clé du récit.

✍️ Rédaction guidée

Deux types de rédaction : Récit (raconter une histoire) ou Texte argumentatif (donner son avis). Sélectionne le type puis dicte dans chaque zone.

💡 Conseils de rédaction

Chaque zone = un paragraphe. Saute une ligne entre chaque partie.
Ne commence jamais par "il", "elle", "car", "parce que".
Utilise des connecteurs : D'abord... Ensuite... Puis... Enfin... En conclusion...
Cite le livre : mets le titre en italique ou entre guillemets.
Relis-toi à voix haute pour repérer les erreurs.

✅ Synchronisation active — Code famille :

Le Tri avec l'IA envoie tes notes de carnet à Claude (Anthropic) pour les classer automatiquement dans les bons onglets.
Pour que ça fonctionne, tu as besoin d'une clé API Anthropic.

Comment obtenir une clé API :
1. Va sur console.anthropic.com
2. Crée un compte → API Keys → Create key
3. Recharge 5$ de crédits (usage familial = plusieurs années)
4. Copie la clé — elle commence par sk-ant-

La clé est stockée uniquement dans ton navigateur (localStorage). Elle n'est jamais envoyée ailleurs que vers l'API Anthropic.

Étape 1 — Créer un projet Firebase gratuit
1. Va sur console.firebase.google.com
2. Clique Ajouter un projet, donne-lui un nom (ex. axel-lecture)
3. Désactive Google Analytics (pas utile) → Créer le projet
Étape 2 — Activer Firestore
1. Dans le menu à gauche → Firestore Database
2. Clique Créer une base de données
3. Choisis Démarrer en mode test → Suivant → Activer
Étape 3 — Récupérer la config
1. Dans le menu à gauche → Paramètres du projet (⚙️)
2. Descends jusqu'à Tes applications → clique </> Web
3. Donne un nom (ex. axel-app) → Enregistrer l'application
4. Copie le bloc JSON entre accolades { apiKey: …, appId: … }
Étape 4 — Connecter l'appli
1. Reviens sur l'onglet ⚙️ Configuration
2. Choisis un code famille simple (même code sur tous tes appareils !)
3. Colle la config Firebase dans la zone de texte
4. Clique Connecter ✅

💡 Sur les autres appareils : refais juste l'étape 4 avec le même code famille.
